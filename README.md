# HR Analytics Dashboard

Interactive HR Analytics Dashboard built using Microsoft Excel.

## Dashboard Preview

![Dashboard](HR_Analytics_Dashboard.png)

## Features
- KPI Cards
- Interactive Slicers
- Pivot Tables
- Attendance Analysis
- Performance Analysis
- Risk Distribution
- Overtime Analysis
- Top 10 Performers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
